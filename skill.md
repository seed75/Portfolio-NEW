## 🎨 Design System (색상 & 디자인 정책)

> Luzia 포트폴리오 참조 — 다크모드 없이 흰 배경 / 검은 폰트로 통일

### Color Palette

| 역할 | 값 | 용도 |
|------|-----|------|
| Background | `#ffffff` (white) | 모든 페이지 배경 |
| Foreground | `#171717` (zinc-900) | 제목, 강조 텍스트 |
| Body text | `#71717a` (zinc-500) | 본문, 설명 텍스트 |
| Muted | `#a1a1aa` (zinc-400) | 부제목, italic 강조 |
| Border | `#e4e4e7` (zinc-200) | 카드 테두리, 구분선 |
| Surface | `#f4f4f5` (zinc-100) | 이미지 플레이스홀더, 태그 배경 |
| Accent (CTA) | `#18181b` (zinc-900) | 버튼 배경 |
| Accent text | `#ffffff` (white) | 버튼 텍스트 |
| Green (status) | `#10b981` (emerald-500) | Available 뱃지 |

### Typography

| 요소 | 크기 | 두께 | 특이사항 |
|------|------|------|---------|
| Hero H1 | `text-5xl` ~ `text-6xl` | `font-semibold` | italic accent: zinc-400 |
| Section H2 | `text-3xl` ~ `text-4xl` | `font-semibold` | |
| Card H3 | `text-base` | `font-semibold` | |
| Body | `text-lg` | `font-normal` | zinc-500, leading-relaxed |
| Label/Tag | `text-xs` ~ `text-sm` | `font-medium` | uppercase tracking-widest |

### Rules
- **다크모드 완전 비활성화** — `globals.css`에서 `prefers-color-scheme: dark` 제거
- `html`, `body` 모두 `bg-white text-zinc-900` 명시
- 모든 컴포넌트에서 `dark:` prefix Tailwind 클래스 사용 금지
